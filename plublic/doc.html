Introducción

Infraestructura Pública

Nexy ofrece acceso simplificado a herramientas de descarga e IA mediante endpoints públicos. No se requiere autenticación mediante API Keys, permitiendo una integración directa en tus proyectos.

Configuración

Base URL

Todas las solicitudes deben dirigirse al siguiente endpoint principal. Asegúrate de incluir el prefijo de api para mantener compatibilidad:

https://api-nexy.ultraplus.click/api
Ejemplo de Uso GET

Petición Estándar

Usa cualquier cliente HTTP o Fetch API para obtener datos en formato JSON. Aquí un ejemplo funcional:

fetch('https://api-nexy.ultraplus.click/api/dl/ytdl?url=URL_VIDEO') .then(res => res.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error));
Respuesta

Formato JSON

Todos los endpoints devuelven respuestas en formato JSON estandarizado con los siguientes campos:

{ "status": 200, "creator": "Nexy", "result": { "title": "Lana Del Rey - Say Yes To Heaven (Official Audio)", "channel": "Lana Del Rey", "type": "audio", "format": "mp3", "quality": "128", "duration": "3:29", "thumbnail": "https://i.ytimg.com/vi/MiAoetOXKcY/hqdefault.jpg", "download": "https://api-nexy.ultraplus.click/cdn/say-yes-to-heaven.mp3" } }
